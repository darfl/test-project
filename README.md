# 💸 Fair Split — умный сплиттер совместных расходов

Приложение для честного и быстрого разделения общих трат в любой компании.  

Забудьте про мучительный подсчёт «кто кому и сколько должен» в заметках телефона — Fair Split сделает всё за вас за пару кликов.

## Для чего это нужно

### 🍽️ Рестораны и бары
Вы пришли большой компанией, кто-то заказал стейк, кто-то — пару коктейлей, а кто-то вообще только десерт. Общий счёт оплатил один человек. Fair Split моментально посчитает, кто и сколько должен организатору.

### 🍕 Совместные позиции на столе
Пицца на двоих, сет роллов на троих, кальян на четверых — всё, что делится не поровну на всех, а только на часть участников. Выберите, кто участвовал — и Fair Split сам распределит доли.

### 🏡 Загородный дом с баней
Вы сняли коттедж на выходные. Аренда оплачена одним участником, продукты — другим, баня — третьим. Fair Split соберёт все чеки воедино и выдаст одну понятную картину: кто кому переводит и сколько.

### 🏔️ Горнолыжная поездка
Билеты на подъёмники, скипассы, общий трансфер, ужины в шале — расходы разбросаны по разным картам и кошелькам. Приложение оптимизирует переводы так, чтобы количество транзакций было минимальным.

### 🚗 Совместные путешествия
Бензин, платные дороги, отели, билеты на экскурсии — Fair Split работает с любыми категориями общих расходов. Создайте событие, добавьте чеки, разделите позиции между нужными участниками — и наслаждайтесь поездкой.

## Личная история

Это приложение родилось из нашей потребности. Мы обожаем большие поездки с друзьями: аренда загородного дома на Новый год, спонтанные поездки в горы на лыжах, шумные ресторанные вечера большой компанией.

И каждый раз после такого отдыха начинался один и тот же ритуал — приходилось составлять сложные таблицы в Excel, уточнять «кто именно курил кальян?», «а пиццу на двоих мы заказывали с Сашей?», «а ты уже скинула деньги или это было в прошлый раз?». Это утомляло.

Мы написали Fair Split для себя — чтобы закрыть эту потребность раз и навсегда. Делитесь им с друзьями, пусть ваши поездки будут наполнены радостью, а не подсчётами.

## Возможности

- ✅ Несколько чеков в одном событии
- ✅ Позиции, разделённые между выбранными участниками
- ✅ Плательщик для каждой позиции
- ✅ Оптимизация количества переводов (greedy-алгоритм)
- ✅ История событий с хранением в localStorage
- ✅ Копирование напоминалки о долгах в буфер обмена
- ✅ Отметка «Оплачено» для каждого долга
- ✅ Отзывчивый дизайн — удобно и на телефоне, и на ноутбуке

## API (основной эндпоинт)

### POST /api/split/calculate

```json
// Запрос
{
  "organizerName": "Алина",
  "sharedItems": [
    {
      "name": "Кальян",
      "amount": 1200,
      "paidBy": "Саша",
      "sharedWith": ["Алина", "Настя", "Никита"]
    },
    {
      "name": "Пицца",
      "amount": 800,
      "paidBy": "Никита",
      "sharedWith": ["Никита", "Саша"]
    }
  ]
}

// Ответ
{
  "average": 500.0,
  "total": 2000.0,
  "debts": [
    { "debtor": "Настя", "creditor": "Саша", "amount": 400.0 },
    { "debtor": "Алина", "creditor": "Саша", "amount": 100.0 },
    { "debtor": "Саша", "creditor": "Никита", "amount": 400.0 }
  ]
}
```

## Запуск

```bash
# Бэкенд (порт 8080)
cd backend && ./mvnw spring-boot:run

# Фронтенд (порт 5173)
cd frontend && npm run dev
```

Откройте http://localhost:5173

## Планы на будущее

Нам хочется, чтобы Fair Split стал по-настоящему народным приложением. Вот что в планах:

- 🗄️ **База данных** — пока все события живут в localStorage вашего браузера. Планируем перевести хранение на сервер, чтобы ничего не терялось при смене устройства.
- 🔐 **Авторизация** — личный кабинет, вход через Telegram и Google, возможность делиться событием с друзьями по ссылке.
- 🤖 **Телеграм-бот** — чтобы добавить чек можно было прямо из мессенджера. Скинул фотографию или написал сумму — бот сам посчитает долги и пришлёт итог в чат.
- 📱 **Мобильное приложение** — React Native-версия для iOS и Android, чтобы считать расходы можно было прямо за столом, не открывая ноутбук.
- 🎨 **Собственные иконки и айдентика** — нарисовать фирменный стиль и добавить анимированные микро-взаимодействия, которые будут радовать глаз.
- 🧪 **Покрытие тестами** — unit- и e2e-тесты для бэкенда и фронтенда, чтобы ничего не сломалось даже при самом неожиданном сценарии.
- 🌍 **Мультиязычность** — русский, английский и, может быть, еще пара языков, чтобы приложением могли пользоваться в разных странах.
- 📊 **Статистика и аналитика** — красивые графики расходов по месяцам, категориям, участникам. Понимать, на что уходит больше всего.
- 🔗 **Интеграция с банками** — автоматический импорт чеков из мобильных приложений банков, чтобы не вбивать суммы вручную.
- 🧾 **Сканирование чеков** — навели камеру, распознали позиции, и Fair Split сам предложил, как разделить.
- 📤 **Экспорт в PDF и Excel** — для тех, кто любит, чтобы всё было задокументировано.
